# Data Manifest

## Raw Data

Place source recipe documents in:

```text
data/raw
```

Supported in the foundation scanner:

- `.docx`
- `.pdf`

Unsupported files are skipped and not included in the inventory.

## Processed Data

Generated files are written to:

```text
data/processed
```

Current generated file:

```text
data/processed/recipes_inventory.csv
```
