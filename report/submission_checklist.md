# Submission Checklist

## Include

- `README.md`
- `report/final_report.md`
- `report/presentation_script.md`
- `report/presentation_bullets.md`
- `report/live_demo_plan.md`
- `demo/run_demo_query.py`
- `demo/live_demo_notebook.ipynb`
- `demo/live_demo_notebook_notes.md`
- key `eval/*.md` findings used in the report
- source code needed to run the demo

## Exclude from manual submission bundle

- `.env`
- `PLAN.md`
- `PROJECT_CONTEXT.md`
- `TASKS.md`
- `eval_files_after_archive.txt`
- `eval_files_after_archive_clean.txt`
- old `report/report.md` if not needed
- virtual environments
- cache folders
- `__pycache__/`
- `.ipynb_checkpoints/`

## Demo

Recommended command:

```text
python demo\run_demo_query.py --question "מה כמות הקמח, המים והשמרים בבצק הפיצה?"
```

Notebook:

```text
demo/live_demo_notebook.ipynb
```

Backup HTML:

```text
demo/demo_output.html
```

Note:
`demo/demo_output.html` is generated by running the demo script. Decide whether to include it as a backup artifact or regenerate it locally before presenting.
